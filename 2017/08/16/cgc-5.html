CSSSR Garbage Collector #5

Aug 16, 2017 by felix_exter
2 minute read

Еженедельная подборка ссылок из чатов CSSSR на интересные материалы.

  • babel-plugin-transform-imports — плагин для Babel, заменяющий именованные импорты на импорты по конкретным путям:
    • пишем по-старинке
      import { reduxForm } from 'redux-form';
    • получаем на выходе
      import reduxForm from 'redux-form/es/reduxForm'
  • babel-plugin-ramda — плагин для Babel, удаляющий неиспользуемые зависимости из Ramda:
    • пишем по-старинке
      import R, { map } from 'ramda';

  map(R.add(1), [1, 2, 3]);
    • получаем на выходе
      import add from 'ramda/src/add';
  import map from 'ramda/src/map';

  map(add(1), [1, 2, 3]);
  • babel-plugin-lodash — плагин для Babel, удаляющий неиспользуемые зависимости из Lodash:
    • пишем по-старинке
      import _ from 'lodash';
  import { add } from 'lodash/fp';

  const addOne = add(1);
  _.map([1, 2, 3], addOne);
    • получаем на выходе
      import _add from 'lodash/fp/add';
  import _map from 'lodash/map';

  const addOne = _add(1);
  _map([1, 2, 3], addOne);
  • react-loadable — компонент высшего порядка для загрузки компонентов с динамическим импортом
  • @hocs — коллекция компонентов высшего порядка для React, особенно полезных в сочетании с Recompose

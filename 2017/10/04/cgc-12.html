CSSSR Garbage Collector #12

Oct 4, 2017 by felix_exter
1 minute read

Еженедельная подборка ссылок из чатов CSSSR на интересные материалы.

  • CSSSR Shorts — короткие обучающие видео на тему React
  • Всегда используй rel="noreferrer noopener", ссылаясь на другой сайт с помощью target="_blank"
  • webpack-blocks — функциональные строительные блоки для твоего webpack конфига: простой способ настроить webpack и делиться конфигурацией между проектами
  • Повторение — мать учения: простое объяснение из рекомендаций от W3C, почему на кнопках нет и не должно быть курсора руки по умолчанию — cursor: pointer обозначает наведение курсора на ссылку
  • Gatsby — сверхбыстрый статический генератор сайта для React, на котором, кстати, недавно редизайнулся сам сайт React
  • en Size Limit: сделай веб легче — демонстрация простой библиотеки, у которой ужали бандл для браузера в 5000 раз
